    Алгоритм
    Быстрая сортировка.

	Разработанный в 1960 году Тони Хоаром алгоритм использовался для сортировки слов в словаре для перевода и сопоставления уже отсортированному словарю. Слава пришла к алгоритму, когда он был включен в Unix как алгоритм сортировки по умолчанию и занял свое почетное место в STL языка C, где и получил свое всемирно известное название qsort.

Start code
public class QuickSortExample {
    public static void main(String[] args) {
        initArray();
        printArray();
        quickSort();
        printArray();
    }
}
End code

	Алгоритм предназначен для поиска элемента в отсортированном по определенному ключу массиве. Сортированные структуры обычно применяются в словарях, где надо найти, например, значение слова, или телефонных книгах. Бинпоиск находит место нужного значения в массиве по его ключу.
